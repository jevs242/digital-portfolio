Experiences

This game was created for a game jam with a team of five people, and it was my first time participating in a game jam. We decided to create a shooter game and had only one week to develop it. As the only programmer in the team, I was responsible for completing the entire game. This experience taught me how to code faster and how to integrate blueprint and C++ in Unreal Engine. I also gained knowledge in collider and level design. The game is available on itch.io, and we won second place in the game jam.

Information

Our team participated in a one and a half week-long gamejam, where we had to create a cyberpunk-style game using a specific color palette. Our game is a short story where the player character goes to a bar to eliminate a boss, but during the fight, they also have to deal with smaller enemies.
