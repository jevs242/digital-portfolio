KrazyRoad

Experiences

For the class portfolio, I developed a game to expand my portfolio and acquire new skills. My goal was to learn how to use Unity, a popular game development engine, and create an endless runner game for smartphones. I chose this type of game because it's a popular genre in the mobile gaming industry, and I wanted to gain experience in this field. Additionally, I wanted to incorporate a shop and ads into the game to further enhance my skills. Over the course of one month, I successfully achieved all my objectives for the game, including integrating ads. This proved to be a significant challenge, but I was able to overcome it through persistence and hard work. After completing the class, I released the game on Google Play, adding another skill to my portfolio. Overall, this project allowed me to acquire new skills, expand my portfolio, and learn more about game development for mobile devices.

Information

This project is an endless runner made Unity was programmed with c #. All assets used in this game are part of the Unity store.
