Run Jimmy Run

Experiences

As part of a challenge to myself, I recently designed and created a 2D game in Unity in just one day. The game was specifically designed for web format, allowing players to easily access and enjoy the game online. Through this project, I was able to further develop my skills in C# programming and game development using Unity. Despite the time constraints, I was able to create a fun and engaging game that provided a valuable learning experience. This project has inspired me to continue exploring the possibilities of game development and to further improve my skills in this field.

Information

"Run Jimmy Run" is an endless runner game that challenges players to overcome three different types of obstacles, each requiring a unique set of actions. The game's fast-paced gameplay and increasing difficulty provide an exciting and engaging experience for players. To successfully navigate through the obstacles, players must master the specific actions required for each one. The game's dynamic design and responsive controls make it an enjoyable and addictive gaming experience. Overall, "Run Jimmy Run" is a thrilling and challenging game that offers hours of fun and entertainment.
